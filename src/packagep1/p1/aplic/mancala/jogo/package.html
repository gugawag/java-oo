As regras do jogo de Mancala. Este pacote contém tudo que se precisa para jogar (jogadores automáticos, tabuleiro, etc.) mas sem interface com o usuário. Um jogo com interface a caractere pode ser construído usando o package p1.aplic.mancala.cui. Um jogo com interface gráfica pode ser construído usando o package p1.aplic.mancala.gui.
